Details

30 Days was the first Game Jam I ever participated in, so it holds a special place in my heart (although the code is likely very messy...). The game was developed solo in 72 hours.

The theme of the jam was "apocalypse", so the game involved upgrading a ship to send humanity to a new planet before Earth gets destroyed. All the art of the game was made in Inkscape, which is a vector-based program similar to Illustrator.
